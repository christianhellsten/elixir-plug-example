# elixir-plug-example

An annotated Elixir and Plug example web application.

```bash
$ git clone https://github.com/christianhellsten/elixir-plug-example.git
$ brew install elixir
$ mix deps.get
$ iex -S mix
```

[http://localhost:4000](http://localhost:4000)
